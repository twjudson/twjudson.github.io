If there is an element \(1 \in R\) such that \(1 \neq 0\) and \(1a = a1 = a\) for each element \(a \in R\text{,}\) we say that \(R\) is a ring with unity or identity. A ring \(R\) for which \(ab = ba\) for all \(a, b\) in \(R\) is called a commutative ring. A commutative ring \(R\) with identity is called an integral domain if, for every \(a, b \in R\) such that \(ab = 0\text{,}\) either \(a = 0\) or \(b = 0\text{.}\) A division ring is a ring \(R\text{,}\) with an identity, in which every nonzero element in \(R\) is a unit; that is, for each \(a \in R\) with \(a \neq 0\text{,}\) there exists a unique element \(a^{-1}\) such that \(a^{-1} a = a a^{-1} = 1\text{.}\) A commutative division ring is called a field.
