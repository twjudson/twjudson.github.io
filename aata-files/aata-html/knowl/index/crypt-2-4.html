Systems that use two separate keys, one for encoding and another for decoding, are called public key cryptosystems. Since knowledge of the encoding key does not allow anyone to guess at the decoding key, the encoding key can be made public. A public key cryptosystem allows \(A\) and \(B\) to send messages to \(C\) using the same encoding key. Anyone is capable of encoding a message to be sent to \(C\text{,}\) but only \(C\) knows how to decode such a message.
