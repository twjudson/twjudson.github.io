Lemma 13.1.7.

A finite abelian group is a \(p\)-group if and only if its order is a power of \(p\text{.}\)

Proof.

If \(|G| = p^n\) then by Lagrange’s theorem, then the order of any \(g \in G\) must divide \(p^n\text{,}\) and therefore must be a power of \(p\text{.}\) Conversely, if \(|G|\) is not a power of \(p\text{,}\) then it has some other prime divisor \(q\text{,}\) so by Lemma 13.1.6, \(G\) has an element of order \(q\) and thus is not a \(p\)-group.
