Example 16.5.2.

Let us solve the system
\begin{align*} x & \equiv 3 \pmod{4}\\ x & \equiv 4 \pmod{5}\text{.} \end{align*}
Using the Euclidean algorithm, we can find integers \(s\) and \(t\) such that \(4s + 5t = 1\text{.}\) Two such integers are \(s = 4\) and \(t = -3\text{.}\) Consequently,
\begin{equation*} x = a + k_1 m = 3 + 4k_1 = 3 + 4[(5 - 4)4] = 19\text{.} \end{equation*}
