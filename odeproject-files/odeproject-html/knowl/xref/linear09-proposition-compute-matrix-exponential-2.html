Proposition 3.9.8.

If \(A\) is an \(n \times n\) matrix with a single eigenvalue \(\lambda\text{,}\) then there exists a nonnegative integer \(k \lt n\) such that
\begin{equation*} e^{tA} = e^{\lambda t} \left( I + t(A - \lambda I) + \frac{t^2}{2!} (A - \lambda I)^2 + \cdots + \frac{t^k}{k!} (A - \lambda I)^k \right). \end{equation*}
