Theorem 1.5.7.

If
\begin{equation*} y' + p(t) y = g(t) \end{equation*}
is a differential equation such that \(y(t_0) = y_0\text{,}\) and \(p(t)\) and \(g(t)\) are continuous on the open interval \(I = (\alpha, \beta)\text{,}\) then there exists a unique function \(y = \phi(t)\) satisfying the differential equation and the initial condition on \(I\text{.}\)

Proof.

If
\begin{equation*} \mu(t) = \exp\left( \int p(t) \, dt \right), \end{equation*}
then
\begin{equation*} \frac{d}{dt}(\mu(t) y) = \mu(t) \left( \frac{dy}{dt} + p(t) y \right) = \mu(t)g(t). \end{equation*}
Integrating both sides of this equation and solving for \(y\text{,}\) we have
\begin{equation} y = \frac{1}{\mu(t)} \left( \int \mu(t) g(t) \, dt + C \right).\tag{1.5.8} \end{equation}
Since \(y(t_0) = y_0\text{,}\) the constant \(C\) is uniquely determined. Notice that we have used continuity to guarantee that the integrals exist. We leave it as an exercise to show that (1.5.8) is a solution to \(y' + p(t) y = g(t)\text{.}\)
